Projects

Sliced

Sliced is an app that dynamically scales recipes. This app is for people who are tired of saying, Hey Siri, what is half of 1/3 cups in tablespoons?

When a user creates a recipe on Sliced, the text is parsed to separate the amount and measurement unit from the ingredient's name. Users can use the slider at the top of the recipe page to scale all the ingredients up and down between 1/4 and 2 by increments of 1/4.

Amounts divisible by 1/8 and 1/3 are converted into fractions, and other amounts are rounded to the third decimal place.

Spaced Repetition

Spaced Repetition is a language learning app that uses the spaced repetition technique to generate a series of Latin vocabulary words.

The frequency of each word is determined by its memory value, which is doubled each time the word is answered correctly and reset to 1 when the word is answered incorrectly. This is implemented with the linked list data structure. The current word is always the head of the list, and words are moved back in the list based on their memory value.

This project was created using Test Driven Development at Thinkful's Engineering Immersion program.

Init

Init is a social media platform for developers to connect over their work.

Users can create posts, comment on posts, follow other users and view the posts of users you follow. This project features a paginated infinite scroll used in the profile and feed components and binary encoded images stored in the PostgreSQL database.
