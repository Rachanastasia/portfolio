Sliced

Sliced is an app that stores and scales recipes. When a user creates a recipe on Sliced, the text is parsed to seperate the amounts and measurement units from the ingredient's name. Users can use the slider at the top of the screen to scale the recipe up and down between 1/4 and 2 by increments of 1/4. Amounts divisible by 1/8 are converted into fractions, and other amounts are rounded to the third decimal place.

Spaced Repetition

Spaced Repetition is a language learning app that uses the spaced repetition technique to generate a series of Latin vocabulary words. The frequencey of each word is determined by it's memory value, which is doubled each time the word is answered correctly and reset to 1 when the word is answered incorrectly. This is implemented with the linked list data structure. The current word is always the head of the list, and words are moved back in the list based on their memory value.

Init

Init is a social media platform for developers to connect over their work. Users can create posts, comment on posts, follow other users, receive notifications, and scroll infinitely through the posts of users you follow. This project features a paginated infinite scroll used in the profile and feed components and binaray encoded images stored in the PostgreSQL database. Init was created by Rachel Reilly, Adyceum Magna Ccarri, Trevor J Alt, and Steven Henderson.
